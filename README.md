<!-- ABOUT THE PROJECT -->
**SFML Chess With AI (In-Progress)**
<p align="center"> **2 player chess game built using the SFML (Simple and Fast Multimedia Layer) Library that also has an AI. You can choose to play against a human or a bot. You can click on the pieces of the board/your color to move and play the game. **
  AI Features:

-Alpha Beta Pruning

-Quiescence Search

-Basic Tapered Evaluation

-Evaluation

-Piece Square Tables

-Transposition Tables (Zobrist Hash)**
  
</p>
